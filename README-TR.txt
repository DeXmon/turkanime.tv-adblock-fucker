turkanime.tv sitesi için Adblock uyarısı kaldıran javascript kodu.
Kullanımı:
  1- Chrome için Tampermonkey veya Firefox için Greasemonkey uzantısını yükleyin.
  2- Tampermonkey için; Chrome sayfasının sağ üstündeki Tampermonkey simgesine tıklayın ve "Add a new script..." seçeneğini seçin.
  3- adblockfucker.js dosyasındaki satırları kopyalayıp yapıştırın ve sol taraftan "Save" butonuna tıklayın. (Uzantının aktif olmasına dikkat edin)
